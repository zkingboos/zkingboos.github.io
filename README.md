# Showcase Site

Personal showcase site built with React, Vite, and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Technologies

- React
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
